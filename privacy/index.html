Privacy Policy for Spend Habit

Last Updated: June 2026

Overview

Spend Habit is a personal finance application that allows users to track expenses, income, and budgets. This Privacy Policy explains how user data is collected, stored, and used.

Information We Collect

Account Information

  • Email address
  • First name
  • Last name
  • Password (securely hashed, never stored in plain text)
  • Account creation date

Financial Data

  • Expenses
  • Income records
  • Budget data
  • Categories

How We Use Data

We use this data only to provide core app functionality, including:

  • User authentication
  • Expense tracking
  • Budget management
  • Data synchronization and storage

We do not sell or share user data with third parties.

Data Storage

All data is securely stored on servers hosted by Hetzner.

Account Deletion

Users may request deletion of their account and all associated data at any time.

To request deletion, please contact: angelmariiano@icloud.com

Once verified, we will delete:

  • Account information
  • All expenses, income, and budgets
  • All associated user data stored on our servers

Data Retention

Data is retained as long as the account is active or as needed to provide the service.

Security

We apply reasonable security measures to protect user data from unauthorized access or disclosure.

Children’s Privacy

Spend Habit is not intended for users under 13 years of age.
